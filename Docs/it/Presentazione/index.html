ReKat:

Guida di installazione

GitHub

Per prima dobbiamo ottenere il codice sorgente.

Il metodo più veloce è tramite GitHub

Installato il programma possiamo scaricare il codice.

Aggiungiamo una Repository con il seguente percorso https://github.com/ELDRON64/ReKat

CMake

Ottenuto il codice dobbiamo trovare un modo per eseguirlo.

A questo scopo ci arriva in aiuto CMake.

CMake genera un particolare tipo di file chiamato Makefile

Questo file ha lo scopo di dare ad un compilatore tutte le istruzioni che gli servono per creare l'eseguibile.

MinGw

Però CMake non lavora a da solo. Ha bisogno di un compilatore, nel nostro caso utilizzeremo un compilatore per il linguaggio c/c++

Per i sistemi Windows utilizzeremo MinGw

Per i sistemi Unix utilizzeremo gcc

Preparazione

Prima di poter compilare il programma dobbiamo preparare il nostro Computer:

Pre iniziare dobbiamo aggiungere CMake e il compilatore a PATH ( per i sistemi Windows ).
Cerchiamo sul menù di ricerca: PATH, e apriamo il programma "Modifica le Variabili di ambiente per l'account"

selezioniamo PATH e Aggiungiamo 2 nuovi valori:
  Il percorso per MinGW
  Il percorso per CMake

Configurazione

Per Configurare il programma usiamo i seguenti passi:

Apriamo il programma CMake-gui

Scegliamo la cartella dove abbiamo scaricato il codice sorgente e la cartella dove vogliamo installare il programma

Premiamo su "Configurate" e selezioniamo MinGW Makefiles con default compilers

Finita la configurazione possiamo generare il progetto con "Gemerate".

Compilazione

Ora possiamo finalmente compilare il programma

Ora apriamo un terminale di Powershell: Win + powershell e eseguiamo i seguenti comandi:
  $ cd "cartella di instalazione"
  $ mingw64-make

Con questi comandi abbiamo creato il programma e ora possiamo eseguirlo.

Utilizzo

Per avviare il programma nella stessa powershell che abbiamo usato prima digitiamo:
  $ cd Build/ReKat
  $ .\ReKat

con questo abbiamo avviato il programma.
